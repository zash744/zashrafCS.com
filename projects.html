  • Meal Tracker Mobile App

    • This mobile application makes it easier to log and track their meals, especially if the meals they consume are repetitive. It allows users to take a picture of their meal and the app will recognize the meal and log specific nutritional data for them. Designed a Flutter/Dart app integrating an Image Recognition API to recognize meals, recording 150+ events and providing key insights into glucose levels for health monitoring.

  • Full Stack Fantasy Football Stat Web Application

    • In this web application, a user can see all the important stats regarding a player in fantasy football to help them draft their team and win their league! This application uses React for the front end and Java Spring Boot for the backend to provide fantasy football players with access to approximately 12,000 NFL statistics, utilizing data scraping and storing information in a MySQL database.

  • Full Stack Web Application for Streaming Availability

    • In this web application, a user can enter any show or movie and this web application will tell the user what platform the show or movie can be found on and will store their entries in a database. So many people hear about new shows or movies but don't know if they are able to watch it platforms that they may subsribe to. With this, they can find out easily. This was done in HTML, CSS, Javascript(Node.js), and MongoDB.
